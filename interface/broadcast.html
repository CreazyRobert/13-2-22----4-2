//发送一条动态(广播)
网址　http://www.halfeet.com/_broad_cast/_send_broad_cast.php?jsonfrom=
参数　{"fc":"send_broad_cast","loginid":"120627-201459-FB4-5AD4-7FD9F2A1A0DA","says":"base64后的json数据"}
返回　{"fc":"send_broad_cast","ok":"y"}

loginid为用户登录ID
replyid为正在回复的该条动态的ID
says为嵌套数据，值为动态(广播)的具体内容
　　-> saytext值为正文的文字信息
　　-> attach值为嵌套信息，是本条动态的附件列表
　　　　-> type值为image/sound等
　　　　-> content值为附件文件的完整URL网址
例：{"saytext":"hello, 我是冠军", "attach":[]}
例：{"saytext":"hello, 我是冠军", "attach":[{"type":"image", "content":"http://xxxxxx/abcd.jpg"}, {"type":"sound", "content":"http://xxxx/abc.mp3"}]}
ok值为y表示返回，为n表示无返回
//返回一条动态(广播)及回复列表

网址　http://www.halfeet.com/_broad_cast/_get_broadcast_byid.php?jsonfrom=
参数　{"fc":"get_broadcast_byid.","loginid":"120627-201459-FB6B7994-F5FB-E1F6-5AD4-7FD9F2A1A0DA","castid":"113199230"}
返回　{"fc":"get_broadcast_byid","ok":"y","mtime":"26分钟前","mlong":"118331400","mlat":"40199230","uid":"1060","pname":"袁俊","uface":"192.168.1.104/photo-user-face/2012_07_30_16/120730-164752-4934BDA3-8D77-D90E-15DA-45548731433E_c.jpg","content":"base64后的json数据","comments":"0","replylist":[]}

loginid值为用户登录ID
castid值为动态广播的ID

ok值为y表示返回成功，值为n表示返回不成功
content值为嵌套数据
　　-> saytext值为正文的文字信息
　　-> attach值为嵌套信息，是本条动态的附件列表
　　　　-> type值为image/sound等
　　　　-> content值为附件文件的完整URL网址
-> pname是personname缩写，值为对方用户的昵称
-> uface是userface缩写，值为对方用户的头像完整路径，含http://
-> mtime是messagetime缩写，值为动态发生的时间
-> mlong是messagelong缩写，值为动态发生的经度坐标
-> mlat是messagelat缩写，值为动态发生的纬度坐标
-> comments是本条动态的被评论次数

replylist值为嵌套数据
　　->mtime值为回复时间
　　->uid值为回复人的ID
　　->pname值为回复人昵称
　　->uface值为回复人的头像完整路径，含http://
　　->content值为嵌套数据，同上
　　　　-> saytext值为正文的文字信息
　　　　-> attach值空，回复限文本内容，不含多媒体
例：{"mtime":"7分钟前","uid":"1060","pname":"袁俊","uface":"http://192.168.1.104/photo-user-face/2012_07_30_16/120730-164752-4934BDA3-8D77-D90E-15DA-45548731433E_c.jpg","content":"eydzYXl0ZXh0Jzon6L+Z5piv55S755qE56We6ams77yfJywnYXR0YWNoJzpbXX0="}

返回　{"ok":"y"}

loginid为用户登录ID
replyid为正在回复的该条动态的ID
says为嵌套数据，值为动态(广播)的具体内容
　　-> saytext值为正文的文字信息
例：{"saytext":"hello, 我是冠军"}

ok值为y表示返回，为n表示无返回
