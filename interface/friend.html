//好友关系变更
网址　http://www.halfeet.com/_friend/_link_user_touser.php?jsonfrom=
参数　{"fc":"link_user_touser","loginid":"120627-201459-FB6B7994-F5FB-E1F6-5AD4-7FD9F2A1A0DA","linktouid":"1115","action":"link"}
返回　{"fc":"link_user_touser","ok":"y"}

loginid值为用户登录id
linktouid值为对方用户的ID（非loginid）
action值为设置关系，link表示加关注，unlink表示取消关注

ok值为n表示执行失败，值为y表示执行成功
好友关系：n陌生人 f粉丝 g关注 h好友 m密友
//所有好友的动态一览
网址　http://www.halfeet.com/_friend/_get_friendaction_ofmy.php?jsonfrom=
参数　{"fc":"get_friendaction_ofmy","loginid":"120627-201459-FB6B7994-F5FB-E1F6-5AD4-7FD9F2A1A0DA"}
返回　{"fc":"get_friendaction_ofmy","ok":"y","list":[]}

loginid值为用户登录id

ok值为n表示执行失败，值为y表示执行成功
list值为嵌套数据
->actid值为动态ID
->mtime值为动态发生的时间，mlong，mlat为动态发生的经纬度
->ltime值为用户最后在线时间，llong，llat为用户最后所在的经纬度
->dist_mmeter动态发生地点与我的距离，dist_lmeter好友当前与我的距离
->disttime到达动态发生点的距离，交通所需时间
->uid为好友的ID半步编号，pname好友的名字
->gender性别f女m男，oldyears年龄，hbody身高，sstar星座，xblood血型
->uface头像文件URL
->content值为嵌套数据
　　-> saytext值为正文的文字信息
　　-> attach值为嵌套信息，是本条动态的附件列表
　　　　-> type值为image/sound等
　　　　-> content值为附件文件的完整URL网址
->comments本条动态的评论数量
//按类别好友的列表
网址　http://www.halfeet.com/_friend/_get_friendlist_ofmy.php?jsonfrom=
参数　{"fc":"get_friendlist_ofmy","loginid":"120627-201459-FB6B7994-F5FB-E1F6-5AD4-7FD9F2A1A0DA","kind":"关注"}
返回　{"fc":"get_friendlist_ofmy","ok":"y","list":[]}

loginid值为登录成功时返回的唯一标识
kind值为与我的关系，关注/粉丝/好友

ok值为y表示返回成功，值为n表示返回不成功
list值为嵌套数据
-> loginid是loginid缩写，值为对方用户的登录ID
-> uid是userid缩写，值为对方用户的用户ID
-> pname是personname缩写，值为对方用户的昵称
-> uface是userface缩写，值为对方用户的头像完整路径
-> ltime是logintime缩写，值为对方的最后更新时间，llong值为对方用户的经度坐标，llat值为对方用户的纬度坐标
-> distmeter是用户与我的距离 　　 -> disttime是用户与我的距离需花费的时间
-> xblood值为血型，hbody值为身高，sstar值为星座，sayme值为个人签名档
-> serveron值为对方用户登录所在服务器

-> distmeter是用户与我的距离 　　 -> disttime是用户与我的距离需花费的时间
