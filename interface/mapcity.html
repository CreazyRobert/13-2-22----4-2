//根据省，县/区关键字检索，返回地址列表

网址　http://www.halfeet.com/_map_city/_get_addresslist_bykey.php?jsonfrom=
参数　{"fc":"get_addresslist_bykey","loginid":"120730-175005-0A12DFC6-E43D-43F3-EC96-3C756AC2E085","province":"北京","town":"朝阳区","keyname":"地铁"}
返回　{"fc":"get_addresslist_bykey","ok":"y","list":[{"mapid":"775889","city":"北京市","town":"朝阳区","name":"城建地铁地基市政工程公司","plong":"116576393","plat":"39956396"},{"mapid":"778059","city":"北京市","town":"石景山区","name":"地铁古城家园","plong":"116187761","plat":"39916783"}],"townlist":[{"town":"东城区","count":"14"},{"town":"丰台区","count":"20"},{"town":"大兴区","count":"1"}]}

loginid值为登录成功时返回的唯一标识
province值为省份名称（直辖市的省和市同名，比北京市是直辖市，下面只有一个市，即北京市，再下面是县和区）
town值为县和区的名称， 若不指定town值，表示搜索整个省份内/直辖市，限返回20条，若指定town值，则不限返回的数量
keyname值为关键字，比如"王府井"，"地铁站", "中行"

ok值为y表示返回成功，值为n表示返回不成功
list值为嵌套数据
-> mapid值为地理名称的ID
-> city值为地理名称所在城市
-> town值为地理名称所在县/区
-> name是pointname缩写，值为地理名称的全称
-> plong是poslong缩写，值为地理坐标经度
-> plat是poslat缩写，值为地理坐标纬度
townlist值为嵌套数据
-> town值为县/区的名称
-> count值为符合条件的数据
